## 🎯 核心玩法

你是一名孤胆战士，在竞技场中迎战一波又一波的敌人。每一波都比上一波更强、更多。你能撑到第几波？

### 操作方式

**PC 端**
- 🖱️ 鼠标瞄准 + 左键射击
- ⌨️ WASD 移动
- 💨 空格键闪避
- 🔄 Q 键切换武器

**手机端**
- 🕹️ 虚拟摇杆移动（左下角）
- 👆 右半屏触控瞄准 + 自动射击
- 💨 滑动手势闪避 / 点击闪避按钮
- 🔄 点击切换武器按钮

## 👥 特色机制：好友复活

这是本游戏最独特的社交机制！

**场景**：你在第 18 波被 BOSS 击败，差一点就能上钻石段位...这时候好友复活就派上用场了！

**流程**：
1. 阵亡界面点击 **"邀请好友复活"**
2. 系统生成专属链接（每个链接对应一次复活机会，防止滥用）
3. 把链接发给好友（微信/QQ/复制链接都可以）
4. 好友在**任何设备**上（手机/电脑）点击链接进入游戏
5. 你点击 **"核验分享"**，系统自动检测好友是否已进入
6. 核验通过 → **满血复活 + 3 秒无敌护盾 + 50 金币 + 5 钻石** 🎉

好友也会获得 **50 金币**的新人奖励！双赢！

> 💡 这个机制支持**跨设备**！你在电脑上玩，好友在手机上点链接，一样可以复活。数据通过云端数据库实时同步。
