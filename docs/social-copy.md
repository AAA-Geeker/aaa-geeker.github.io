## 文案使用建议

1. **配图更重要**：分享时务必附带游戏截图或录屏 GIF
2. **带上个人战绩**：分享链接带上自己的最高分/段位更有吸引力
3. **时间选择**：晚上 8-10 点和周末下午是游戏社区活跃时段
4. **互动引导**：文案末尾加"来挑战我吧"比"来玩吧"转化率高 3 倍
5. **多平台差异化**：开发者社区强调技术，游戏社区强调玩法，朋友圈强调社交
