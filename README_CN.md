# Survival Arena - 生存竞技场

一款功能丰富的俯视角波次生存射击游戏，使用原生 HTML5 Canvas 构建，可直接部署到 GitHub Pages。

## 立刻游玩

直接用现代浏览器打开 `index.html`，或部署到 GitHub Pages 即可开始游戏。

## GitHub Pages 部署

1. 将此文件夹推送到 GitHub 仓库
2. 进入 Settings → Pages
3. 将来源设置为 `main` 分支，根目录
4. 保存 — 你的游戏就上线了！

## 游戏特色

### 核心玩法
- **WASD 移动** + **鼠标瞄准**，自动射击
- **空格键冲刺**，用于紧急闪避
- **5 种敌人类型**：普通兵、奔跑者、重装兵、射击兵和 Boss
- **每 5 波出现 Boss**，Boss 拥有弹幕攻击模式
- **屏幕震动**和粒子特效，带来爽快的战斗体验

### 成长系统
- **6 条升级路线**：最大生命值、移动速度、攻击力、射速、金币获取
- **可解锁皮肤**，使用高级货币（钻石）购买
- **每日奖励系统**（7 天连续登录，奖励逐步递增）
- **连击系统** — 连续击杀可获得额外分数和金币加成

### 道具
- ❤️ 生命恢复
- ⚡ 加速
- 💥 双倍伤害
- 🛡️ 护盾
- 🧲 金币磁铁
- 💰 金币袋
- 💎 稀有宝石掉落

### 传播与留存机制
- **好友邀请系统**：分享专属链接，好友通过链接加入可获得 50 额外金币
- **复活令牌**：好友通过你的链接加入时获得
- **死亡时的多种复活方式**：
  - 📺 观看广告（模拟倒计时，免费）
  - 🎫 使用复活令牌（通过邀请获得）
  - 👥 邀请好友复活你
  - 💎 消耗钻石

### 商业化设计
- **双货币体系**：金币（游戏内获取）+ 钻石（高级货币）
- **皮肤商店**：使用钻石购买外观皮肤
- **复活令牌**：可在商店中用钻石购买
- **钻石礼包**：模拟内购（$6 购买 50 钻石 + 2 个复活令牌）
- **广告位**：模拟 15 秒广告换取免费复活

## 自定义配置

- **广告时长**：修改 `js/game.js` 中的 `CFG.AD_DURATION`
- **敌人属性**：修改 `ENEMY_TYPES` 对象
- **商店价格**：编辑 `showShop()` 方法中的价格
- **接入真实广告**：将广告倒计时逻辑替换为你的广告 SDK
- **接入真实支付**：将基于 confirm 的模拟购买替换为 Stripe / 微信支付等

## 技术栈

- localStorage 本地持久化
